Project information

Chat app is built using the socket io, javascript library which provides real time Bi-directional communication between client and server. The app can be used by anyone to join the chat room if the person provide the name and the chat room name.

Tools used
  • HTML5
  • CSS3
  • Websocket io
  • NodeJS
NPM Packages
  • Socket.io
  • Nodemon
  • Express
  • Bad-words
The app is Deployed in Heroku
